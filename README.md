# VisionSpice – Indian Spices Classification

**Dataset:** Indian Spices Image Dataset (19 classes, 10,991 images)

### Models Included
1. **VS_APP1_CNN** – Baseline custom CNN  
2. **VS_APP2_EfficientNet** – Transfer learning (EfficientNet-B0)

### How to Train
1. Prepare dataset in folder structure:  

```
indian_spices/
├── Asafoetida/
├── BayLeaf/
...
```

2. Run notebooks under `notebooks/`.

These results can support research internship applications (NUS SoC, etc.)
