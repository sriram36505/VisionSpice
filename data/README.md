Download the Indian Spices Image Dataset from Mendeley:
[Indian Spices Image Dataset](https://data.mendeley.com/datasets/vg77y9rtjb/3)

## Dataset Statistics

- **Total Classes**: 19 Indian spice types
- **Total Images**: 10,991 high-quality images
- **Image Format**: RGB JPEG
- **Image Resolution**: Various (preprocessing applied in training pipeline)

## Usage

1. Download the dataset from the Mendeley link above
2. Extract the dataset
3. Place the `indian_spices` folder in this `data/` directory
4. Update the dataset path in `config.yaml` if needed
5. Run training scripts from the root directory
